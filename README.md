#  AI-Powered Smart Farming Platform

An intelligent farming assistant built using **Python and Streamlit** that helps farmers detect crop diseases, receive crop advisory, plan mixed cropping, and discover suitable government schemes using AI-driven insights and real-time data.

---

## Features

-  Crop disease detection using plant image analysis  
-  Location-based crop recommendations  
-  Mixed cropping suggestions  
-  Government scheme recommendations  
-  Interactive farmer-friendly dashboard  
-  AI-powered automated insights  

---

## 🛠 Tech Stack

- **Frontend/UI:** Streamlit  
- **Backend:** Python  
- **Machine Learning:** TensorFlow / PyTorch / OpenCV (update if needed)  
- **Data Processing:** Pandas, NumPy  
- **Version Control:** Git, GitHub

##  Installation & Run Locally

### 1 Clone the repository

git clone <your-repo-link>
cd ycce_hackathon

### 2 Run the Streamlit app
streamlit run app.py 

#### Environment Variables
Create a .env file in the root folder and add:
API_KEY=your_api_key_here
